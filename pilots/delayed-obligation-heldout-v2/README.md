# Delayed-obligation held-out extension v2

This prospectively frozen internal synthetic validation extends, but does not modify, `pilots/delayed-obligation-dual-task/`. V1 is development evidence. V2 introduces two unseen forms for each of the same two purposive work shapes and crosses every form with `neutral_interrupt`, `channel_hint`, and `oracle_reminder` (12 cells). The forms require multi-predicate dispositions or quantitative derivations so primary-artifact checks can fail independently of obligation checks.

The harness records each query issuance, subsequent channel availability, returned channel content, public due-cue presentation, and oracle presentation. Encoding and adoption remain **unobserved**; agent state fields are retained only as agent-reported action artifacts. The oracle remains privileged evaluator evidence, not an internal-memory aid. Shapes and conditions are reported cell-by-cell and are not pooled.

Predeclared sequence:

```bash
python pilots/delayed-obligation-heldout-v2/freeze_protocol.py
python pilots/delayed-obligation-heldout-v2/run_study.py preflight
# commit and push the frozen protocol before any model call
python pilots/delayed-obligation-heldout-v2/run_study.py execute
python pilots/delayed-obligation-heldout-v2/run_study.py replay
```

Perfect, planted obligation-failure, and primary-artifact mutation canaries are zero-call checks. No outcome-based retries or selective replacement are allowed. Invalid or service-failed attempts remain in intended/retained denominators. This instrument can support only exact conformance and bounded observations on these configured attempts; it cannot establish a causal treatment effect, agent or professional capability, general prospective memory, expert validity, safety, production fitness, readiness, population representativeness, or cross-domain generality.

## Provenance and design rationale

The obligation-stage design and claim ceilings derive from `papers/agent-benchmarks/2026-07-16-pmbench-prospective-memory-validity.md` and the consolidated v1 evidence in `docs/benchmark-design-taxonomy.md`. V1 showed a 18/18 primary-turn ceiling and used state self-report partly as an access proxy. V2 therefore (1) freezes unseen non-ceiling candidates, (2) adds deterministic primary subchecks and mutation canaries, and (3) records environment-mediated channel flow directly. These two synthetic work shapes test reusable dual-task machinery; they are not domain commitments or expert-authored professional cases.
