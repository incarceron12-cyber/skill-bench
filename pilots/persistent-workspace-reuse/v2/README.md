# Persistent-workspace reuse v2

This internal synthetic validation slice prospectively freezes and executes a **12-cell configured-agent matrix**: two unseen equivalent forms each for structured tables and structured memos, crossed with reset, information-matched full-history, and curated-correct retained-state presentations. Each cell is attempted once. Current requirements, current information, base-artifact availability, model/provider/tools, and budgets are matched; only retained-state presentation changes.

## Charter decision filter

- **Objectives:** B and C: executable expertise-to-evaluation and benchmark infrastructure.
- **Evidence/artifact:** frozen protocol, isolation/authority/compatibility/grader canaries, retained configured-agent attempts, direct artifact/state/usage observations, exact replay, and mutation tests.
- **Uncertainty:** whether the harness can produce comparable retained-state records without conflating deterministic reruns or claiming unobserved semantic adoption.
- **Mode:** building and validation.
- **Scope/duplication:** the two artifact shapes test a cross-domain reuse mechanism; they do not define a procurement or policy benchmark. v1 remains byte-preserved and supplies separate stale/conflicting/revoked and deterministic controls.
- **Useful completion:** all intended attempts are retained once, cells remain separate, hashes/graders replay, and failures fail closed.

## Boundaries

`retained-state.json` differs by condition. Full-history and curated-correct contain the same retained fact and base artifact but present them as chronological messages versus typed current objects. Reset has an empty retained composition. All conditions receive identical current requirements, current information, and base artifact.

Agent-emitted access receipts establish only a declared file-access surface. They do **not** independently establish internal semantic adoption, which remains `unobserved`. Deterministic reruns and stale/conflicting/revoked authority cases remain zero-call preflight controls and are never pooled with agent outcomes.

No result licenses a retained-state causal effect, general memory or agent capability, expert/professional validity, privacy, collaboration, safety, production fitness, readiness, or cross-domain generalization.

## Commands

```bash
python pilots/persistent-workspace-reuse/v2/run_study.py preflight
python pilots/persistent-workspace-reuse/v2/run_study.py execute
python pilots/persistent-workspace-reuse/v2/run_study.py replay
python -m unittest tests.test_persistent_workspace_reuse_v2 -v
```

The protocol and instrument must be committed and pushed before `execute`; retries and replacement attempts are prohibited.
