# Crossed evaluator-validity audit (bounded slice)

This prospective, builder-authored calibration freezes nine cases across a structured evidence memo and an irreversible delegated action. It instantiates the Tool-Veritas review's five-link warning as executable cases: true failure, irrelevant difference, alternate valid path, stale/wrong target, missing observer view, incomplete communication, and unsafe superficial completion.

`evaluator-inputs.json` contains no reference verdict or rationale. `oracle-private.json` is separately authored and loaded only after evaluator execution. `audit.py` crosses a retained exact-target native evaluator and a factual-gate-plus-restricted-qualitative evaluator against that reference, preserves configuration hashes, raw verdicts, abstention, failure-type rows, disagreement loci, sensitivity/specificity, and variance limits.

The unrestricted model-judge and regenerated-rubric conditions are deliberately recorded as `not_executed`: this bounded run did not have a predeclared pinned model configuration and does not fabricate model evidence. Thus the artifact is a **partial protocol/conformance slice**, not the completed crossed empirical audit.

Run:

```bash
python pilots/crossed-evaluator-audit/audit.py
python -m unittest tests.test_crossed_evaluator_audit -v
```
