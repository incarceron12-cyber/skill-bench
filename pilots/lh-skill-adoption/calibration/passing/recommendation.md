# Recommendation

Run a controlled pilot; do not adopt skill-grounded scoring as validated professional measurement yet.

The reported expert-rubric agreement of 0.60 is evidence of scoring consistency, not professional validity [E01]. Individual output concordance was only 0.08 in the reported Figma runs [E05]. The execution evidence was 7 paired runs over 3 tasks and remains directional [E07]. The salient +0.87 Codex delta should motivate the pilot, not settle it [E08].

Adopt only if independently authored and shared-rubric conditions both improve expert readiness without worsening evidence errors; stop if process scores rise without expert readiness or if the effect does not replicate.
