# Multilingual workflow-edge equivalence conformance

This is an **internal deterministic calibration slice**, not a multilingual benchmark or an agent trial. It advances charter objectives B/C by testing reusable meaning-preservation and evaluator-boundary machinery across language-bearing workflow edges without making localization a project scope boundary.

## Design

A single builder-authored supplier-notice fact pack is projected into five matched forms: native monolingual, instruction-swapped, source-swapped, output-swapped, and fully multilingual. The forms freeze semantic facts and authority while changing only declared instruction/source/output language roles and output locale. The contract pins Unicode normalization, font, renderer, and locale-tool identities and declares permitted date/number/unit display transforms.

The replay covers one passing control and five planted diagnoses:

- semantic transfer (negation reversed);
- locale formatting (German output uses the wrong numeric display);
- provenance loss;
- unsupported authority;
- grader-language bias, which yields `insufficient_evidence` rather than a substantive failure.

The invariant keys explicitly cover entity, number, date, unit, negation, modality, scope, authority, and provenance. `diagnostic-report.json` is generated by `validate.py`; it is not hand-authored outcome evidence.

## Provenance and limits

Design rationale comes from `papers/agent-benchmarks/2026-07-12-polyworkbench-multilingual-workflow-validity.md`, especially **Unique insight**, **Transfer to skill-bench**, and **Concrete changes**. That review reports no inspectable matched PolyWorkBench forms. Therefore this fixture's facts, translations, component identities, and planted outcomes are synthetic contract-calibration material.

No qualified bilingual or domain reviewer participated. Both gates remain explicitly `unmet`. The artifact supports only the validator's exact behavior on these cases; it does **not** support multilingual capability, real cross-language equivalence, professional validity, grader fairness, or deployment readiness.
