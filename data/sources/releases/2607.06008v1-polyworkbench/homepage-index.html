About PolyWorkBench

PolyWorkBench evaluates LLM agents on complex, multilingual, long-horizon tasks requiring multi-step tool use, cross-lingual understanding, and structured output generation.

Evaluation Tracks:

  • Grade — Task-specific multi-dimensional structural scoring (primary metric)
  • Pytest — Functional pass/fail test suite
  • Judge — LLM-as-Judge quality assessment

Current Status: All scores are n=1 (single run). Future updates will include n=3 with Pass@3 and Pass^3 metrics.
