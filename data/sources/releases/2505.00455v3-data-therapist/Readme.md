This OSF repository contains the manuscript, video (+subtitles), and supplementary materials to the experiment.

The compressed folder has three folder: dataset, participants-responses, and prompting-techniques.

In the folder dataset, there are datasets that are used to conduct the experiment.
In the folder participants-responses, there are annotations that the participants have made using the tool, as well as the scores that the participants evaluated.
In the folder prompting-techniques, it contains codes that are used to generate the prompts.

How to read participants-responses (explanation of the features):
This dataset contains a csv file of responses and analysis about the three datasets. 
The columns are:
A1~AX, B1~BX: indices
Similarity/Different: Whether we can find similar content+intent in the counterpart's annotations
Similarity w/: Similar annoations (using index)
Questions: Questions used by DT
Responses: Responses/Annotations
Location: Location in the spreadsheet.

In the file "Evaluation of annotations with visualization designers..pdf", it has the information about how visualization designers evaluated the annotations by trying to find questions, charts, and etc..
